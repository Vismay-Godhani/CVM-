# CVM++

A custom programming language, bytecode compiler, and stack-based virtual machine built from scratch in C++.

## Features

- Interactive REPL
- Lexer and tokenizer
- Recursive-descent parser
- AST generation
- Bytecode compiler
- Stack-based virtual machine
- Variables and expressions
- Boolean logic
- Conditional statements
- While loops
- Debug mode

## Compilation Pipeline

```text
Source Code
    ↓
Lexer
    ↓
Tokens
    ↓
Parser
    ↓
AST
    ↓
Compiler
    ↓
Bytecode
    ↓
Virtual Machine
    ↓
Output
```

## Build

```bash
g++ Main.cpp -std=c++17 -o cvm
```

## Usage

Run REPL:

```bash
./cvm
```

Run a script:

```bash
./cvm program.cvm
```

Run with debug information:

```bash
./cvm program.cvm --debug
```

## REPL

```text
CVM++ REPL
Type 'exit' to quit.

>> x = 10
>> y = 20
>> print x + y
30
```

Exit with:

```text
exit
```

or

```text
quit
```

## Language Syntax

### Variables

```cvm
x = 10
name = "Vismay"
flag = true
```

### Print

```cvm
print x
print "Hello World"
```

### Input

```cvm
print "Enter age:"
age = input
```

### Arithmetic

```cvm
10 + 5
10 - 5
10 * 5
10 / 5
10 % 3
```

### Comparisons

```cvm
10 == 10
10 != 5
10 > 5
10 < 20
10 >= 10
10 <= 20
```

### Boolean Logic

```cvm
a && b
a || b
!a
```

### If Statement

```cvm
if x > 5:
    print "Greater"
```

### If Else

```cvm
if x > 5:
    print "Greater"
else:
    print "Smaller"
```

### While Loop

```cvm
count = 5

while count > 0:
    print count
    count = count - 1
```

## Example Program

```cvm
x = 10
y = 20

sum = x + y

print sum

if sum > 25:
    print "Large"
else:
    print "Small"

count = 3

while count > 0:
    print count
    count = count - 1
```

## Architecture

### Lexer

```cvm
x = 10 + 20
```

↓

```text
IDENTIFIER
ASSIGN
NUMBER
PLUS
NUMBER
```

### Parser

```text
ASSIGN
├── x
└── ADD
    ├── 10
    └── 20
```

### Compiler

```text
PUSH_INT 10
PUSH_INT 20
ADD
STORE x
```

### Virtual Machine

```text
PUSH 10
PUSH 20
ADD

Stack: [30]
```
